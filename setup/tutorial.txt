Para iniciar:

    criar tabelas no banco de dados no arquivo models.py(todas as classe criadas devem herdar
    da classe-pai 'Model')

Serialização:

    a Serialização é o processo de transformar os models em JSON's. para criar um serializer, é
    necessário criar classes que herdem da classe-pai 'ModelSerializer'. temos de criar, dentro
    da classe Serializer, uma classe 'Meta' atribuir aos atributos model e fields os valores 
    correspondentes.
